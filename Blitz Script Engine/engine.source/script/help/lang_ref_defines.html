Defines

Defines are labels that have fixed values, strings, or macros that will not change (ever) during the course of your program. These labels are replaced during compile time.  

The Define keyword is used to assign an identifier to a constant. For example:

Define one_hundred=100

You can then use the identifier 'one_hundred' anywhere in your program instead of '100'.

There are a few rules to consider when using Defines:

1. They must be declared at the top of the main program file.

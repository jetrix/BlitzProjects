Keywords 

The following keywords are built into BlitzScript, and may not be used as identifiers (variables, function names, labels, etc.):


And, Case,  Data, DefaultCase, Define,  Else, ElseIf, End, EndIf, Exit, False, For, Function, If, Next, Not, Null, Or,  Read, Rem, Return, Sar, Select, Step, Then, To, True, While, Xor, Include, Push, Pop, Jump

. Keywords are not implemented at this time.

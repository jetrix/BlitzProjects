Arrays 

Arrays are created using the standard BASIC 'Dim ' statement, and may be only one dimension. For example:

Dim arr[10]

Creates an array called 'arr' with 11 elements numbered 0...10.

An array may be dimensioned at more than one point in a program, each time an array is dimensioned, it's previous contents are discarded. Arrays may be dimensioned inside functions, but a corresponding 'Dim' statement of the same array must also appear somewhere in the main program. For example:

Dim test[0]

Function Setup( x )

Dim test[x]

End Function

BlitzScript automatically creates a Pointer for arrays so they can be treated like a LIFO Stack. Push , Pop, and Jump Functions can be used to store and retreive values.

Push(test[x]) ;increments pointer by one, assigns value into to element

x = Pop(test) ;assigns value at element into x, decrement pointer by one

Jump test(x) ;positioned test array pointer at x
